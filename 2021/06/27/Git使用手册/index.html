GitHub个人搭建过程

参考：https://hexoscript.gitbook.io/hexo-script/v/1.3-1/shi-yong-bu-zhou/zhun-bei-gong-ju

一、建立个人GitHub账号，并建立个人博客页面仓库

1.首先在GitHub网站创建个人账号，此处省略（自行解决，毕竟创建账号并不是什么难题）。在此处提供一个好用的VPN—佛跳墙。

2.创建个人仓库，进入GitHub，点击Repositories–》New，

repository name 写：Masuo777.github.io，替换成自己的名字。

其他的按照自己的意思来即可。

二、安装所需环境以及配置SSH

1.node.js

下载地址：http://nodejs.cn/download/

建议使用安装程序进行安装。安装时，请勾选Add to PATH选项。

2.git

Git官网下载：https://git-scm.com/

国内镜像下载：https://npm.taobao.org/mirrors/git-for-windows/v2.22.0.windows.1/Git-2.22.0-64-bit.exe

国内其他版本下载：https://npm.taobao.org/mirrors/git-for-windows/

四、配置自己的博客样式

想要更多各式各样的主题可以在（https://hexo.io/themes/)
这里查看。找到它的可下载或clone链接，下载到本地，只需要修改hexo下的_config.yml中的theme字段。

打开hexoblog文件下的themes，如果自己clone 的主题没有放在这里，可以将自己下载的主题中的相同文件夹（themes）下的文件copy过来。

并修改自己的config.yml文件中的theme为copy过来的文件名。

此时重启服务(hexo s)，每次修改文件后想要其生效都需要重启服务。

即可看到自己想要的样式的博客了。

六、修改个人博客具体样式

以上已经将博客搭建完成，想要修改具体的博客样式可以参照以下方法。

查看插入的theme下的REDEME.md文件，在这个文件中，这个theme的作者会告诉我们具体有哪些属性可以修改，以及修改的地方及方式。

例，我的博客引入的是cyan style，在这个文件夹下有REDAME.md,他告诉我需要修改_config.yml。

  • 修改导航栏

导航栏更改需要在首先在config.xml下的menu字段中添加要添加的字段，与导航栏中的字段对应，

之后需要在/source/下面添加对应的目录。

  • 修改侧边栏

要编辑侧边栏需要在’xxx\themes\cyan style\layout’下面找相关文件

  • 修改文章显示

让博客文章只显示部分，剩余的内容以“更多内容”显示
