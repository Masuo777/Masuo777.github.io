2、值得写法

字面量：普通的值（数字，字符，布尔）

​ k:v ：字面直接来写（字符串默认不用加上单引号或双引号

​ “”：双引号，不会转义字符串里面的特殊字符，特殊字符为本身想表达的意思

​ name:”zhangsan \n lisi”：输出：zhangsan 换行 lisi

​ ‘’：单引号，会转义特殊字符，特殊字符最终只是一个普通的字符串数据

​ name:’zhangsan \n lisi’：输出：zhangsan \n lisi

对象、Map（属性和值，键值对）：

​ k: v;（注意空格）

500错误：服务器内部错误（Internal server error）大多是由于服务器引起的错误，不是客户端的错误，即是由于网页内部的错误，网页内部也就是语法，引用等错误。

2、***Mapper的引用错误

1）有可能是因为mapper文件没有正确配置

注解Mapper如果在主程序体没有MapperScan的情况下一定要有，主程序体有Mapper Scan的情况下可以不要，Repository可有可无，没有Repository的情况下，引入Mapper文件时会爆红线，但是对程序运行无影响，加上之后红线消失。

2）可能是因为Mapper文件里的方法重名，这样在对Mapper文件进行扫描时会报错

虽然函数方法不一样，但在对函数register时会产生错误。所以尽量避免在同一个mapper类中有重名函数。

3、***Mapper的Error creating bean with name ‘clubMapper’ defined in file

这个是因为类型不被包含，需要你仔细检查bean对象中的类型是否正确。

在引用数据库字符类型时，字符数据类型大多来自java.util类型
